# QuestBoard
App designed to give out random daily, weekly and monthly quests out of a customizable pool.


So, basically I've tried using different habit making apps, but I end up giving up on them because making the exact same "quests" daily gets very boring, so I decided to make try and make an app that does that but randomly assigning the daily habits, with also weekly and monthly "quests", just like a videogame! So that way I don't get that bored doing chores and they exist like little main quests someone else asigns for me uwu.
